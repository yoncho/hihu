## HiHu | 안녕 한서대 🤗  

한서대학교 대학 생활 도움 서비스     
**HiHu**는 Hi ! Hanseo University의 약자이다.    
  
### 1. 기능

1. 학생 정보 조회 
2. 학과 정보 조회 
3. 학사 일정 조회
4. 통학 버스 예매 및 결제                   
5. 대학 주변 식당 정보 조회 및 전화 연결  
6. 대학 주변 자취방 정보  
7. 지역버스 시간 정보 
8. 동아리 정보  
  
### 2. 서비스 기간
**2019.02.28 ~ 2021.01.09**
  
### 3. 총 사용자
서비스 기간 동안, 총 누적 방문자 13,153명   
학기중, 하루 평균 방문자 400~500명
